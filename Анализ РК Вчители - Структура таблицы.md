# eCademy аналіз реклами для HR - Структура таблицы

## Назначение таблицы
Таблица для анализа рекламных кампаний по привлечению преподавателей с детальным отслеживанием лидов через весь воронку от собеседования до трудоустройства в качестве учителя.

## Столбцы и их описание

### Основная информация о кампании
- **A: Назва реклами** - Название рекламной кампании (текст)
- **B: Посилання на рекламну компанію** - URL ссылка на рекламный аккаунт в Facebook Ads Manager
- **C: Дата аналізу** - Дата проведения анализа (формат: ДД,М,РРРР)
- **D: Період аналізу запущеної компанії** - Диапазон дат работы кампании (формат: ДД,ММ,РРРР - ДД,ММ,РРРР)
- **E: Витрачений бюджет в $** - Потраченный бюджет в долларах с форматированием
- **F: Місце знаходження (країни чи міста)** - Страна или регион таргета (UA, EUR, DE, Other)

### Базовые данные по лидам
- **G: Кількість лідів** - Общее количество полученных лидов
- **H: Кількість лідів перевірка** - Количество проверенных лидов

### Лиды в процессе обработки (В процесі опрацювання)
- **I: Не розібрані ліди (в процесі опрацювання)** - Неразобранные лиды
- **J: Взяті в роботу (в процесі опрацювання)** - Взяты в работу
- **K: Контакт (в процесі опрацювання) ЦА** - Установлен контакт (целевая аудитория)
- **L: НЕ дозвон (в процесі опрацювання) НЕ ЦА** - Нет дозвона (не целевая аудитория)
- **M: Співбесіда (в процесі опрацювання) ЦА** - Собеседование назначено (целевая аудитория)
- **N: СП проведено (в процесі опрацювання) ЦА** - Собеседование проведено (целевая аудитория)
- **O: Не з'явився на СП (в процесі опрацювання) ЦА** - Не явился на собеседование (целевая аудитория)
- **P: Завуч затвердив кандидата (в процесі опрацювання) ЦА** - Завуч утвердил кандидата (целевая аудитория)
- **Q: Завуч не затвердив кандидата (відмовився) ЦА** - Завуч отказал (целевая аудитория)
- **R: Переговори (в процесі опрацювання) ЦА** - В переговорах (целевая аудитория)
- **S: Стажування ЦА** - На стажировке (целевая аудитория)
- **T: Не має учнів ЦА** - Нет учеников (целевая аудитория)
- **U: Вчитель ЦА** - Учитель (целевая аудитория) - финальный статус

### Категории потерь и отказов
- **V: Втрачений (відмовився) ЦА** - Потеря (отказ) целевая аудитория
- **W: Резерв стажування (в процесі опрацювання) ЦА** - Резерв стажировки (целевая аудитория)
- **X: Резерв дзвінок (в процесі опрацювання) ЦА** - Резерв перезвонить (целевая аудитория)
- **Y: Офбординг (відмовився) ЦА** - Офбординг, отказ (целевая аудитория)
- **Z: Звільнився (відмовився) ЦА** - Уволился, отказ (целевая аудитория)
- **AA: Втрачений не цільовий (не цільовий) НЕ ЦА** - Потеря не целевой (не целевая аудитория)
- **AB: Втрачений недозвон (не цільовий) НЕ ЦА** - Потеря по недозвону (не целевая аудитория)
- **AC: Втрачений не актуально (не цільовий) НЕ ЦА** - Потеря (не актуально, не целевая аудитория)
- **AD: Втрачений мала зп (відмовився) ЦА** - Потеря по малой зарплате (целевая аудитория)
- **AE: Втрачений назавжди (не цільовий) НЕ ЦА** - Потеря навсегда (не целевая аудитория)
- **AF: Втрачений перевірити Вайбер (не цільовий) НЕ ЦА** - Потеря (нужно проверить Viber, не целевая аудитория)
- **AG: Втрачений ігнорує (відмовився) ЦА** - Потеря (игнорирует, целевая аудитория)

### Итоговые счетчики
- **AH: Кількість прийшов на співбесіду** - Количество пришедших на собеседование (формула)
- **AI: Кількість які не потрапили в Бот ТГ** - Количество не попавших в Bot Telegram
- **AJ: Кількість "відмовився" загалом** - Общее количество отказов (формула)
- **AK: Кількість (в процесці опрацювання) загалом** - Общее количество в обработке (формула)
- **AL: Кількість на етапі "Стажування"** - Количество на этапе стажировки (формула)
- **AM: Кількість цільових лідів** - Количество целевых лидов (формула)
- **AN: Кількість не цільових лідів** - Количество нецелевых лидов (формула)

### Метрики конверсии (%)
- **AO: Конверсія відмов (%)** - Процент отказов (формула)
- **AP: Конверсія в опрацюванні (%)** - Процент в обработке (формула)
- **AQ: Конверсія з ліда у СП (%)** - Конверсия из лида в собеседование (формула)
- **AR: Конверсія з ліда у стажера (%)** - Конверсия из лида в стажера (формула)
- **AS: Конверсія з прийшов на співбесіду в стажування** - Конверсия из пришедших на СП в стажировку (формула)
- **AT: % цільових лідів** - Процент целевых лидов (формула)
- **AU: % не цільових лідів** - Процент нецелевых лидов (формула)

### Финансовые показатели
- **AV: Ціна в $ за ліда** - Стоимость одного лида (формула: Бюджет / Кількість лідів)
- **AW: Ціна в $ за цільового ліда** - Стоимость одного целевого лида (формула: Бюджет / Кількість цільових лідів)

### Статус кампании
- **AX: Статус рекламної кампанії** - Статус (варианты: "Працює" (Работает), "Спостерігаємо" (Наблюдаем), "Відключили" (Отключили), "Інше" (Другое))

## Структура данных

### Специальные строки
- **Строки 2-3**: Примечания ("2 раза перепроверил", "1 раза перепроверил")
- **Строка 4-11**: Первый набор кампаний с итоговой строкой "Загалом" (строка 12)
- **Строка 13**: Пустая
- **Строка 14-21**: Повторный набор кампаний с итоговой строкой "Загалом" (строка 22)
- **Строка 23**: Пустая
- **Строка 24-25**: Новая серия кампаний "TEST_prepod_UKR_08.04_newcreo" и "TEST_prepod_09.12" с итоговой строкой "Загалом" (строка 26)

- На собеседование: 37
- Проведено СП: 2
- На стажировке: 2

- Конверсия из лида в СП: 37%

## Особенности таблицы

1. **Воронка продаж для HR**: Таблица отслеживает полный цикл от получения лида до трудоустройства учителя
2. **Множественные группировки**: Данные организованы в группы по датам анализа с итоговыми строками
3. **ЦА/не ЦА разделение**: Четкое разделение на целевую и нецелевую аудиторию
4. **Множество причин отказов**: 7+ вариантов потерь и отказов на разных этапах
5. **Статусы кампаний**: Позволяет отслеживать состояние каждой кампании (активна, на паузе, отключена)

## Обозначения
- **ЦА** - Целевая аудитория (подходящие кандидаты)
- **не ЦА** - Не целевая аудитория (неподходящие кандидаты)
- **СП** - Собеседование
- **Завуч** - Администратор/руководитель школы
- **Стажування** - Стажировка
